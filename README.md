# Get to Know Me Through the Films I Watch

## Description

This is a simple website created for a Web Programming assignment. The website introduces myself through several films that I enjoy watching. The idea is that films can reflect personality, perspective, and interests.

## Website Content

The website consists of two pages:

**Home Page**
This page contains a short personal introduction and several films that represent my personality and perspective.

**My Films Page**
This page displays a table of films I like, including their posters, genres, and short explanations. It also includes a form where visitors can recommend movies and briefly describe them.

## Technologies Used

* HTML for creating the structure of the pages
* CSS for styling the layout, colors, and appearance

## Files

* `home.html` – main page with introduction and film highlights
* `page2.html` – page containing the film table and recommendation form
* `styles.css` – stylesheet used to design the website
